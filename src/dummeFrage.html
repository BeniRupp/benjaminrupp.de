Was ist der Unterschied zwischen URI und URL?

Eine URI (Uniform Resource Identifier) identifiziert eine logische oder physikalische Resource welche von Webtechnologien verwendet wird.

Eine URL (Uniform Resource Locator) gibt an, wo eine Ressource im Internet zu finden ist. Im Kontext von HTTP spricht man dann von einer "Website" oder einem "Link". URLs können aber auch zum Übertragen von Dateien (ftp) oder Versenden von E-Mails (mailto) verwendet werden.

Eine URL ist eine URI.
